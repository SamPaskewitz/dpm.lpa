outcome analysis

We now want to see if the latent profiles are related to some outcome variables. We first use the ‘outcome_analysis_normal’ function to run the analysis. This returns a list with various results. We save this list as ‘result’. You can view the function’s help file (using ‘?outcome_analysis_normal’) for a description of each of the elements contained in the function’s output. We are mainly interested in two outputs: the results of analysis using Bayes factors, and a plot of estimated profile outcome means.
